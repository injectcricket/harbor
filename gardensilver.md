# NexusLink

NexusLink 是一个面向全球体育数据聚合与深度链接导航的开源基础设施项目。项目定位于为体育数据分析师、赛事运营团队、开发测试人员以及量化投研爱好者提供统一、可扩展的赛事情报源入口与结构化外链管理方案。通过声明式的资源配置与轻量级调度核心，NexusLink 解决了多源异构赛事数据发现效率低下、链接维护成本高昂以及测试环境数据孤岛等实际问题，可作为自建数据中台的链路前置层或独立情报聚合服务。

## 功能概览

- **多源赛事链接聚合管理**：提供统一的链接注册与分类存储接口，支持批量导入不同地区、不同级别赛事的官方数据页面与实时信息源。

- **动态源健康度探针**：内置轻量级 HTTP 状态检测与响应时间记录机制，可定时验证聚合链接的可达性，并输出可用性报告。

- **声明式资源编排**：支持通过 YAML 或 JSON 配置文件声明链接属性、更新频率、标签体系与依赖关系，便于版本控制与自动化运维。

- **标签化检索与过滤**：为每条链接提供多维标签支持，可按赛事类型、地区、数据层级、语言等维度进行快速筛选与组合查询。

- **外部数据快照缓存**：支持对部分稳定链接进行结构化元数据缓存，减少对上游源站的重复请求，提升内部访问效率。

- **标准化输出适配器**：提供 JSON、CSV、Markdown 表格等多种输出格式，便于对接下游数据分析流水线、监控看板或文档站点。

- **可嵌入的轻量级管理面板**：基于 Web 的只读管理界面，支持链接列表展示、状态标记和快速跳转，适合内部团队使用。

## 应用场景

- **赛事数据中台前置链接层**：数据工程师可将 NexusLink 部署为内部数据中台的统一外链管理模块，集中维护赛事数据来源，避免因人员变动或文档缺失导致的链接失联问题。

- **测试环境数据源模拟**：测试团队可利用 NexusLink 聚合的多个赛事链接作为外部依赖的替代数据源，配合缓存机制构建稳定可控的测试数据集，降低对生产第三方接口的依赖。

- **量化投研情报采集**：投研团队可将 NexusLink 作为情报采集系统的起始节点，定期同步所配置的赛事链接列表，自动化抓取赛事进程、排名变动等信息，辅助模型决策。

- **赛事运营可视化看板**：运营团队可基于 NexusLink 输出的结构化链接数据，快速搭建赛事动态看板，实现赛事入口的一站式跳转与状态监控。

- **开源数据导航站点**：个人开发者或社区可基于 NexusLink 快速构建特定体育领域的开源导航站点，将分散的赛事资源进行有序整理与公开分享。

## 快速开始

以下步骤指导您在本地环境中快速克隆、安装依赖并启动 NexusLink 基础服务。

```bash
# 1. 克隆项目仓库
git clone https://github.com/nexuslink-io/nexuslink-core.git
cd nexuslink-core

# 2. 安装依赖（使用 pip 与虚拟环境）
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 3. 初始化配置并启动服务
cp config/example.yaml config/local.yaml
python main.py --config config/local.yaml --mode server
```

## 常见问题

**问：NexusLink 是否支持动态添加链接而无需重启服务？**

答：支持。项目提供了配置热加载机制，当检测到配置文件变更时会自动重新加载链接列表并更新调度队列。您也可以通过管理面板的只读接口触发手动刷新。需要注意的是，热加载不会影响正在进行的探测任务，新配置将在下一轮调度周期生效。

**问：如何扩展自定义探针以支持非 HTTP 协议？**

答：您可以通过实现 `nexuslink.probe.base.BaseProbe` 抽象类来创建自定义探针，并重写 `check` 方法。完成后，在配置文件中通过 `probe.type` 字段注册您的探针类路径。项目内置的复合探针还支持将多个基础探针串行或并行组合，满足更复杂的验证需求。

**问：聚合链接的可用性数据如何持久化与查询？**

答：每次探针执行后的结果（包括状态码、响应时间、错误信息）会默认写入 SQLite 数据库的 `probe_history` 表中。您可以通过存储模块提供的 `repository` 接口按时间范围、链接 ID 或状态进行过滤查询。输出适配器也支持将历史统计结果导出为报告文件，便于离线分析。
